# UP42

A geospatial data marketplace and aggregator API by Airbus that unifies tasking and archive ordering across many providers — Pléiades Neo (30cm), Pléiades, SPOT, Vantor (formerly Maxar), Capella, ICEYE, BlackSky, Planet SkySat, Wyvern hyperspectral, Vexcel aerial, and more — behind a single console, REST API, and Python SDK.

**Tier:** Paid (credit-based, no subscription required)
**Auth:** OAuth 2.0 (project-scoped credentials)
**Last researched:** 2026-05-07

## Use cases

- Single-API access to commercial high-resolution and very-high-resolution optical, SAR, hyperspectral, and aerial imagery across vendors.
- Tasking workflows across multiple constellations (e.g., compare feasibility from Pléiades Neo, BlackSky, and SkySat for the same AOI).
- Archive catalog search via a STAC-compliant interface, with standardized output formats.
- Mid-market geospatial product teams that need flexible, contract-free access to many providers.

## Pricing

- **Free tier:** No fully free production tier; new accounts can browse the catalog and view sample data without credits. Account creation is free.
- **Paid tiers:** Credit-based — €0.01 = 1 UP42 credit. Buy credits up front (valid 2 years) and spend them across:
  - Archive catalog imagery — pricing varies by provider and resolution; e.g., Pléiades Neo 30cm typically prices in the tens of EUR/km², SPOT around €1.50–€6/km², Sentinel-2 free.
  - Tasking orders — pricing varies by sensor, AOI size, priority tier, and processing options. Minimum AOI charges apply per collection.
  - Processing algorithms from third parties (Airbus, Orbital Insight, Simularity, Nara Space, etc.) — also priced in credits.
- **Enterprise:** Volume contracts and direct sales available for high-throughput customers.
- **Notes:** Pricing is surfaced via API quote endpoints before order confirmation; users always see total cost before commit.

## Authentication

OAuth 2.0 client-credentials. Create a project in the UP42 console, generate `project_id` + `project_api_key`, then exchange for a bearer token. Token sent as `Authorization: Bearer <token>`. The Python SDK abstracts this with a config JSON.

## Endpoints

- **Base URL:** `https://api.up42.com`
- **Key endpoints:**
  - `POST /oauth/token` — token exchange.
  - `GET /catalog/...` — STAC-compliant archive search across data products.
  - `POST /v2/tasking/orders/quotations` — feasibility study + price quote for a tasking order.
  - `POST /v2/tasking/orders` — submit a tasking order.
  - `POST /v2/orders` — submit an archive order.
  - `GET /v2/storage/assets` — list / download delivered assets.
  - `POST /workflows/...` — chain processing algorithms with data orders.

## Example call

```bash
# 1. Get token
TOKEN=$(curl -s -u "$UP42_PROJECT_ID:$UP42_PROJECT_API_KEY" \
  -X POST "https://api.up42.com/oauth/token" \
  -d "grant_type=client_credentials" | jq -r .access_token)

# 2. Search the catalog for Pléiades archive scenes
curl -X POST "https://api.up42.com/catalog/hosts/oneatlas/stac/search" \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "datetime": "2026-04-01T00:00:00Z/2026-05-06T23:59:59Z",
    "intersects": {"type": "Point", "coordinates": [2.349014, 48.864716]},
    "collections": ["phr"],
    "limit": 10,
    "query": {"cloudCoverage": {"lte": 20}}
  }'
```

```python
# Python via the official up42-py SDK
import up42
up42.authenticate(project_id="...", project_api_key="...")
catalog = up42.initialize_catalog()
results = catalog.search(
    {"datetime": "2026-04-01T00:00:00Z/2026-05-06T23:59:59Z",
     "intersects": {"type": "Point", "coordinates": [2.349014, 48.864716]},
     "collections": ["phr"],
     "limit": 10,
     "query": {"cloudCoverage": {"lte": 20}}}
)
```

## Rate limits

Standard fair-use throttling. 429 responses with `Retry-After`. Specific limits not publicly itemized; commercial volume customers get elevated limits via support.

## Notes

- UP42 is owned by Airbus, which means deep integration with the Airbus Pléiades Neo / Pléiades / SPOT / TerraSAR-X catalogs in addition to the third-party constellations on the marketplace.
- For SAR users, UP42 is one of the few platforms that surface Capella, ICEYE, Synspective, Umbra, and Airbus SAR through a unified ordering flow.
- Credits are valid for 2 years from purchase, which is generous compared to monthly subscription models.
- "Live" / latency varies by underlying provider; UP42 mediates feasibility and delivery but doesn't change provider-side processing or archive holds.
- For Airbus-specific data with no marketplace markup, OneAtlas Living Library is a direct alternative; UP42 is the broader / multi-vendor option.
