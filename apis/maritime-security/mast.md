# MAST (Maritime Asset Security and Training)

UK-based maritime security advisory specialising in geopolitical and security advisory services, intelligence-led support, vessel hardening, and embarked PMSC services. Cataloged here as an authoritative Asia/Strait-relevant advisory shop, despite having no public API.

**Tier:** Enterprise
**Auth:** N/A — advisory services delivered via reports, briefings, and direct engagement
**Last researched:** 2026-05-09

## Use cases

- Bespoke geopolitical advisory and threat assessments for clients operating in the Strait of Malacca, Singapore Strait, South China Sea, and adjacent regions.
- Intelligence-led pre-voyage briefings and ad-hoc analyst Q&A.
- Vessel hardening guidance (BMP-aligned for High Risk Areas).
- Embarked maritime security personnel (PMSC).
- Security training for crew (anti-piracy, hostage survival, port security drills).
