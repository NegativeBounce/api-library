# FanDuel

FanDuel is a US-licensed sportsbook and DFS operator with no public developer API; production access to FanDuel odds data goes through licensed aggregators or affiliate-program arrangements.

**Tier:** Enterprise
**Auth:** Partnership only (no public developer portal)
**Last researched:** 2026-05-09

## Use cases

- Production sportsbooks pulling FanDuel-branded lines via licensed aggregators
- Affiliate-program partners tracking referred bettor activity
- Reference odds for odds-comparison and content sites (via aggregators, not direct)

## Pricing

- **Free tier:** None. There is no public FanDuel developer API to subscribe to.
- **Paid tiers:** None publicly documented.
- **Enterprise:** Direct commercial deals are negotiated privately with FanDuel; no public pricing.
- **Notes:** Almost every "FanDuel API" pricing page online sells aggregator access (OpticOdds, SportsDataIO, OddsJam, The Odds API), not direct FanDuel data.

## Authentication

FanDuel does not operate a public developer portal that issues API keys. The internal endpoints used by the FanDuel apps require cookies and session tokens generated by signing in to the consumer app, and using them is contrary to FanDuel's Terms of Use. Affiliate-program participants receive limited credentials scoped to their tracked traffic.

## Endpoints

- **Base URL:** Not publicly documented. App traffic targets internal hosts under `*.sportsbook.fanduel.com` and `*.api.fanduel.com`.
- **Key endpoints:** Not publicly documented.

## Example call

```bash
# No officially supported curl example exists for FanDuel directly.
# Most developers source FanDuel odds via an aggregator, e.g. The Odds API:
curl "https://api.the-odds-api.com/v4/sports/americanfootball_nfl/odds?\
regions=us&bookmakers=fanduel&apiKey=$ODDS_API_KEY"
```

## Notes

- FanDuel does not publish a public developer API. There is no public SDK, no developer portal, no API keys for general use.
- Aggregator access is the only realistic production path. Every major odds aggregator — The Odds API, OpticOdds, SportsGameOdds, OddsJam, SportsDataIO — provides FanDuel as one of their normalized bookmaker keys.
- The FanDuel affiliate program is the closest thing to "official" data access, but it is structured for marketing partners (tracking links, conversion data) rather than for raw odds consumption.
- US-only: FanDuel sportsbook operates only in jurisdictions where it is licensed. Geofencing applies at both the consumer and any data-access layer.
